# Chi Code

基于 Zed 编辑器的中文本地化版本，提供完整的中文界面支持。

## 项目简介

Chi Code 是一个高性能、现代化的代码编辑器，基于 Zed 编辑器深度定制，专注于为中文用户提供更好的开发体验。

### 主要特性

- 完整的中文界面支持
- 高性能 GPU 加速渲染
- 原生 AI 助手集成
- 实时协作功能
- 现代化的 UI 设计
